# 路演讲稿

## 15 秒

大家好，我们是「知乎吹牛逼」。我们做的是一个披着知乎问题外壳的互动游戏：用户点一个自己真的想问的问题，做几道选择题，最后找到一条和自己站在同一边的知乎高赞回声。

## 90 秒

很多人来知乎提问时，并不是真的从零开始找答案。

他心里通常已经偏向一边了，只是还想看看：有没有过来人也这么想？有没有高赞回答能替我把那口气说出来？我这样想，是不是也可以被理解？

所以我们做了「知乎吹牛逼」。首页是一张漂浮的问题星图：要不要裸辞、该不该复合、我炒股能不能赚钱、为什么努力没有结果、我是不是太普通了……

用户点进去后，会连续做几道很轻的选择题。每个选择不是为了算分，而是在判断他更需要鼓励、降温、认同、止损、行动，还是体面地收尾。

最后结果页会给出三件东西：

第一，知乎相似问题和高赞回答链接，让用户找到“原来不止我这样”的圈子。

第二，把高赞回答浓缩成一两句人话。

第三，生成一句可以保存的小卡片。它不是直接告诉用户该怎么办，而是把他心里已经偏向的那句话，换一种更能接住自己的方式说出来。

它的娱乐性来自点击和分叉，它的传播性来自结果卡，它的知乎价值来自真实问题和高赞回答的认同感。我们希望把知乎 15 年的人类经验，变成一个轻量、可玩、能安放情绪的前端应用。

## 3 分钟演示路径

1. 打开首页，展示漂浮问题星图和轮播副标题。
2. 点击「我炒股能不能赚钱？」或「要不要裸辞？」。
3. 连续选择 3 个选项，强调每个选项都像用户心里的真实辩解。
4. 展示结果页：站在哪一边、真正想问的问题、知乎相似高赞、高赞回声、结果判词。
5. 点击复制和保存小卡片。
6. 返回首页，在输入框输入“我想辞职开咖啡店”，展示自定义问题匹配。
7. 说明 OAuth 和知乎 API 的正式接入路径：登录后结合用户授权兴趣和高赞回答，生成更贴近个人处境的结果。

## 一句话收束

「知乎吹牛逼」不是替用户做决定，而是用知乎真实问答，帮他找到那个已经在心里排队的答案。
