# 知乎吹牛逼鉴定局：黑客松提交说明

## 一句话介绍

你能分清知乎高赞、AI 仿深刻和纯吹牛逼吗？

## 产品定位

「知乎吹牛逼鉴定局」不是一个传统灵感生成器，而是一个互动式内容判断游戏。用户在真假难辨的回答盲测中，识别“真洞察”和“一本正经的废话”，最后得到自己的抗吹牛逼指数与人格卡。

## 为什么适合灵感引擎赛道

知乎 15 年社区数据沉淀的不只是内容本身，也是一套中文互联网公共讨论的表达范式。这个 Demo 把“高质量回答为什么像高质量回答”做成可玩的游戏，再把每局揭晓转化成创作者可复用的表达结构。

## 核心玩法

1. 每局展示一个知乎式热议问题。
2. 系统给出三段回答：社区高赞风格、AI 仿深刻、纯吹牛逼。
3. 用户给每段回答盖章鉴定。
4. 揭晓后解释每段回答为什么像高赞、为什么像 AI、为什么是纯吹。
5. 五局结束后生成抗吹牛逼指数、易感话术和可分享结果卡。
6. 结果页将鉴定结果转化为创作者可用的选题角度、高赞开头和评论区钩子。

## 已支持的演示模式

- **普通鉴定局**：随机题库，适合用户自由玩。
- **今日热榜实验室**：从 `data/hot-rounds.json` 读取热榜实验题库，后续可替换成官方 API。
- **评委演示模式**：第一局为路演破冰题，适合现场让评委先参与再讲产品洞察。
- **朋友挑战局**：输入任意话题后生成带 URL 参数的挑战链接，可分享给朋友做同题挑战。
- **创作者灵感转化**：从用户最容易中招的话术反推可写角度，形成灵感引擎闭环。

## 传播机制

- 用户有胜负欲：想证明自己能分清“真懂”和“装懂”。
- 结果有社交货币：抗吹牛逼指数、易感话术、人格标签适合截图。
- 朋友能互相挑战：结果文案天然引导转发。
- 内容能二创：临场造题模块支持输入话题，生成朋友挑战局。

## 路演演示路径

1. 打开首页，强调第一屏就是游戏，不是说明页。
2. 点击「直接看路演局」进入固定题目。
3. 给三段回答盖章，点击「揭晓鉴定」。
4. 展示系统不仅判对错，还解释“为什么这段话骗到你”。
5. 输入一个现场热点话题，生成临场挑战局。
6. 完成或跳转到结果页，展示抗吹牛逼指数和分享卡。

## API 接入设计

当前 Demo 使用本地 fallback 题库保证路演稳定。拿到知乎官方 API 文档后，优先将「今日热榜实验室」替换为实时接口：

- 热榜问题进入每局 `question`
- 高赞回答结构进入 `human` 类型素材
- AI 生成答案进入 `ai` 类型素材
- 纯吹答案由模型根据话术模板生成
- 接口不可用时仍回退到本地题库

## 后续可扩展

- 接入知乎热榜，自动生成每日挑战题。
- 基于真实高赞回答结构训练“高赞风格识别器”。
- 做群聊挑战排行榜，形成多人传播。
- 给创作者输出“高赞结构拆解”和“反吹牛逼修稿建议”。
- 用官方 API 的真实热榜与高赞回答替换当前实验题库。
