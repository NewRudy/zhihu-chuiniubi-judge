# 评审要点说明

## 创新性

- 不是传统心理测试，也不是普通 AI 建议工具，而是把知乎式真实困惑做成选择分叉。
- 用户在点击中认出自己偏向的答案，结果页再用高赞回答式文案给出认同感。
- 前台是轻游戏，后台是问题匹配、情绪状态判断和高赞回声浓缩。

## 知乎契合度

- 入口问题来自知乎最典型的真实提问场景：裸辞、复合、分手、转行、炒股、努力无果等。
- 结果页给出知乎相似问题/高赞回答链接，帮助用户找到“同一派”的讨论圈子。
- 后续可通过 OAuth 和官方 API，把真实高赞回答、热榜问题和用户兴趣接入结果生成。

## 传播潜力

- 问题天然有点击欲：用户会想知道“它最后会怎么说我”。
- 选项像内心独白，用户容易产生“这就是我”的代入感。
- 结果句短、准、适合截图，例如“你不是一定要离开。你只是太久没有感觉到，自己还能选择。”
- 分享卡一键保存，适合朋友圈/群聊传播。

## 落地价值

- 对普通用户：提供轻量情绪出口和认同感。
- 对知乎社区：把高赞问答从搜索消费变成互动消费。
- 对创作者：每条路线都能反推真实困惑、用户立场和内容选题。

## API 接入价值

- 官方 API 可替换当前 fallback 搜索链接，提供真实问题、真实高赞回答和统计数据。
- OAuth 登录后可结合用户兴趣，让推荐的问题和回答更贴近用户。
- 高赞回答摘要可作为“回声”素材，AI 负责压缩和改写，不凭空编造。
